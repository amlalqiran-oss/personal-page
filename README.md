Aml Alqiran - Personal Page
A responsive, professional personal page for Aml Alqiran, a Frontend Web Developer. This project features a modern dark navy blue and gray color palette, smooth animations, and a clean, accessible design.

Preview
The site features a dark-themed, professional layout with:

A sticky navigation bar with a glassmorphism effect on scroll.
A hero section with a staggered entrance animation.
Interactive accordion sections for Services and FAQs.
A responsive grid layout that adapts to mobile screens.
Features
Responsive Design: Optimized for desktop, tablet, and mobile devices.
Modern Aesthetics: Dark navy blue palette with cyan accents and a subtle grid background.
Smooth Animations: Scroll-triggered reveal animations and hover micro-interactions.
Interactive UI: Accordion menus for services and FAQ sections built with vanilla JavaScript.
Accessibility: Semantic HTML5 structure, ARIA labels, and prefers-reduced-motion support.
No Frameworks: Built with vanilla HTML, CSS (Tailwind for utilities), and JavaScript.
Tech Stack
HTML5: Semantic structure.
CSS3: Custom properties (variables), Flexbox, CSS Grid, and keyframe animations.
JavaScript: Intersection Observer for scroll animations, DOM manipulation for accordions.
Tailwind CSS: Loaded via CDN for utility classes.
